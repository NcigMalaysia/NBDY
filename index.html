Penyambungan Google Sheets

Sambungkan sistem ini ke Google Sheet anda secara terus untuk auto-sync setiap kali anda daftar, kemas kini atau padam aduan.

Belum Disambung

Masukkan Web App URL di bawah untuk bermula.

Cara Penyediaan Google Sheet:

  1. Buka Google Sheet baharu anda.
  2. Klik Extensions > Apps Script di menu atas.
  3. Padam kod sedia ada, tampalkan Kod Apps Script di bawah.
  4. Klik butang Deploy > New Deployment.
  5. Klik ikon gear di sebelah Select type, pilih Web app.
  6. Setkan Execute as kepada Me (emel anda) dan Who has access kepada Anyone (Penting untuk sambungan!).
  7. Klik Deploy, berikan kebenaran jika diminta, salin Web App URL yang diberikan, tampal di atas, dan klik Simpan.
